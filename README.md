# Personal Finance Tracker

A beginner-friendly Python project to track expenses and visualize spending using charts.

## Features
- Add daily expenses
- View stored expenses
- Visualize spending pattern using Pie Chart
- Auto-save data in CSV

## Tech Stack
- Python
- Pandas
- Matplotlib

## How to Run
```
pip install -r requirements.txt
python main.py
```

## Files
- main.py — main program
- expenses.csv — auto-generated data storage
- requirements.txt
- README.md
